Для чего Voyage?

Voyage помогает собираться без суеты: списки под города, умные категории, шаблоны для повторных поездок и архив для истории.

  • Создавайте чек‑листы для конкретных поездок.
  • Группируйте вещи по категориям и эмодзи.
  • Сохраняйте наборы как шаблоны.
  • Архивируйте старые шаблоны.

Частые вопросы

  • Как создать чек‑лист? Нажмите «Создать чек‑лист» на главном экране.
  • Как сохранить как шаблон? Откройте список и нажмите «Все собрано».
  • Где архив? В профиле → «Архив шаблонов».
  • Нужна регистрация? Нет, приложение работает в локальном режиме.
